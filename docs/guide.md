# Руководство пользователя — Аптека

## 1. Вход в систему

Откройте браузер и перейдите по адресу: **http://localhost:5173**

Введите логин и пароль. По умолчанию:
- Логин: `admin`
- Пароль: `admin123`

## 2. Dashboard

На главной странице отображается:
- Продажи за сегодня и за месяц
- Количество чеков
- Прибыль
- Топ-10 товаров
- Графики продаж

## 3. Товары

### Добавление товара
1. Нажмите кнопку "Добавить"
2. Заполните форму
3. Нажмите "Создать"

### Импорт из Excel
1. Подготовьте файл XLSX или CSV с колонками: Наименование, Производитель, Штрихкод, Цена и т.д.
2. Нажмите "Импорт" и выберите файл

### Экспорт в Excel
Нажмите кнопку "Экспорт" для скачивания файла XLSX со всеми товарами.

## 4. POS Касса

### Продажа товара
1. Найдите товар через поиск или отсканируйте штрихкод
2. Товар добавится в чек
3. Измените количество при необходимости
4. Добавьте скидку (на товар или весь чек)
5. Нажмите "Оплата"
6. Выберите способ оплаты: наличные или карта

### Штрихкод
Введите штрихкод в поле "Штрихкод" и нажмите Enter.

## 5. Склад

### Приход товара
1. Перейдите на вкладку "Приход"
2. Выберите товар
3. Укажите количество и цену
4. При необходимости выберите поставщика
5. Нажмите "Выполнить"

### Списание
1. Перейдите на вкладку "Списание"
2. Выберите товар и количество
3. Укажите причину
4. Нажмите "Выполнить"

## 6. Инвентаризация

1. Откройте страницу "Инвентаризация"
2. Введите фактические остатки для каждого товара
3. Система автоматически рассчитает разницу
4. Нажмите "Применить" для фиксации результатов

## 7. Контроль сроков годности

Система автоматически отслеживает сроки годности:
- 🟢 **Зелёный** — более 90 дней
- 🟡 **Жёлтый** — менее 90 дней
- 🔴 **Красный** — менее 30 дней или просрочен

## 8. Отчёты

Выберите период (день/неделя/месяц/год) или укажите даты вручную.

Доступные данные:
- Выручка
- Себестоимость
- Прибыль
- Средний чек
- Список продаж

Экспорт в Excel и PDF.

## 9. Управление пользователями

Только администратор может:
- Создавать новых пользователей
- Назначать роли
- Деактивировать учётные записи
- Сбрасывать пароли

## 10. Резервное копирование

Файл базы данных: `database/apteka.db`

Рекомендуется регулярно копировать этот файл в безопасное место.
