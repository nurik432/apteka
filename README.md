# 🏥 Аптека — Система учёта и продаж

Полноценная локальная система управления аптекой. Работает без интернета, все данные хранятся в SQLite.

## 📋 Возможности

- ✅ **Dashboard** — обзор продаж, прибыли, графики
- ✅ **Товары** — полный CRUD, поиск, фильтрация, импорт/экспорт Excel
- ✅ **POS Касса** — интерфейс продаж, штрихкоды, скидки
- ✅ **Склад** — приход, списание, возврат, история движения
- ✅ **Инвентаризация** — сверка остатков
- ✅ **Контроль сроков годности** — цветовая индикация 🟢🟡🔴
- ✅ **Поставщики** — карточки, история поставок
- ✅ **Отчёты** — продажи, выручка, прибыль, средний чек
- ✅ **Аналитика** — графики по дням, месяцам, категориям
- ✅ **Пользователи** — 4 роли с разграничением прав
- ✅ **Тёмная и светлая тема**
- ✅ **Экспорт** — XLSX, CSV, PDF

## 🔧 Требования

- **Node.js** 18.0 или выше
- **npm** 9.0 или выше
- **Windows / macOS / Linux**

## 🚀 Установка и запуск

### 1. Установка зависимостей

```bash
npm install
```

### 2. Настройка базы данных

```bash
npm run setup
```

Эта команда автоматически:
- Создаст файл базы данных `database/apteka.db`
- Выполнит миграции Prisma
- Создаст администратора

### 3. Запуск проекта

```bash
npm run dev
```

Откройте браузер: **http://localhost:5173**

| Роль | Описание |

## 💾 Резервное копирование

Для создания резервной копии скопируйте файл:

```
database/apteka.db
```

Для восстановления — замените файл и перезапустите приложение.

## 📊 Импорт товаров

Поддерживаемые форматы: **XLSX**, **CSV**

Колонки файла (русские или английские названия):
- Наименование / name
- Производитель / manufacturer
- Страна / country
- Штрихкод / barcode
- Артикул / sku
- Закупочная цена / purchasePrice
- Цена продажи / sellingPrice
- Остаток / stock
- Мин. остаток / minStock

## 🛠 Технологии

| Компонент | Технология |

| Frontend | React 19, Vite, TypeScript, Tailwind CSS v4 |
| Backend | Node.js, Express 5, TypeScript |

| Иконки | Lucide React |
| HTTP клиент | Axios |

## ⚠️ Важно

- Все данные хранятся **локально** в файле SQLite
- Приложение работает **без интернета**
- Не используются облачные сервисы
- Пароли хранятся в хешированном виде (bcrypt)
- JWT токены действуют 24 часа
